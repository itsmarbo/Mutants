MutaGen Kv7.2

A free machine learning web service to predict pathogenicity of missense variants in Kv7.2 channels

To run the analysis you need to provide the mutation as follows:

  1. Select the letter of the inicial aminoacid using the slider
  2. Check the number where the mutation takes place
  3. Select the letter of the final aminoacid
  4. Run the Model!
