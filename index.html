MutaGen Kv7.2

A free machine learning web service to predict pathogenicity of missense variants in Kv7.2 channels

HOW TO USE THIS APPLICATION

Just insert a mutation indicating the position followed by the new aminoacid separated by a "." or, by the new nucleotide separated by ":" in case of DNA mutations.
New aminoacids can be indicated in 1 or 3 letter code
FULL-LENGHT PROTEIN IS CONSIDERED, e.i., the first aminoacid is a Methionine
